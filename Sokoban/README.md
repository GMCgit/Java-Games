# Some syntax in the game!

## "#" is a wall
  You can't pass through those!

## "." is background
  Does nothing!

## "☃" is you, the player!
  Try moving it!

## "□" is a block
  You can mobe it by hitting it

## "◎" is a target
  You need to push the block there

## "☇" is a death block
  Do not touch them (Work in progress)
